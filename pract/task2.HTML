Computer programming

From Wikipedia, the free encyclopedia

Computer programming is the process of designing and building an executable computer program for accomplishing a specific computing task. Programming involves tasks such as analysis, generating algorithms and implementation of algorithms in a chosen programming language.

History

Programmable devices have existed at least as far back as 1206 AD, when the automata of Al-Jazari were programmable via pegs and cams to play rhythms. The 1801 Jacquard loom could produce different weaves by changing punched cards.

The first computer program is generally dated to 1843 when mathematician Ada Lovelace published an algorithm to calculate Bernoulli numbers intended for Charles Babbage's Analytical Engine.

Machine code was the language of early programs written directly in the instruction set of the machine. Assembly languages were later developed to make programming easier.

Modern programming

Modern programming includes a wide range of methodologies and tools used to develop software systems efficiently and reliably.

Quality requirements

Whatever the approach to development may be, the final program must satisfy fundamental properties such as reliability, robustness, usability, portability, maintainability and efficiency.

Readability of source code

In computer programming, readability refers to how easily a human can understand the purpose and structure of source code. Readable code helps developers maintain and improve programs over time.

Algorithmic complexity

The academic field of computer programming is largely concerned with discovering and implementing efficient algorithms. Algorithms are often classified using Big-O notation which measures time or memory usage.

Chess algorithms as an example

Programming a Computer for Playing Chess (1950) evaluated the minimax algorithm which became an important concept in algorithmic complexity and artificial intelligence.

Programming languages

Different programming languages support different programming paradigms. Some languages are low-level and close to machine instructions while others are high-level and easier for humans to use.

Programmers

Computer programmers are those who write computer software. Their work usually involves:

  • Coding
  • Debugging
  • Documentation
  • Integration
  • Maintenance
  • Requirements analysis
  • Software architecture
  • Software testing
  • Specification
